欢迎来到我的博客！这是我使用 GitHub Pages 和 Jekyll 创建的第一篇文章。

## GitHub Pages 是什么？

**GitHub Pages** 是 GitHub 提供的免费静态网站托管服务。它非常适合用于：

- 📄 个人网站和博客
- 📚 项目文档
- 📊 作品集展示
- 📖 技术教程

### 为什么选择 GitHub Pages？

1. **免费** - 完全免费使用
2. **简单** - 只需推送代码到 GitHub
3. **自动构建** - 支持 Jekyll 自动构建
4. **自定义域名** - 支持绑定自己的域名
5. **HTTPS** - 自动启用 HTTPS 加密

## Jekyll 简介

**Jekyll** 是一个简单的静态网站生成器。它的工作原理是：

```
Markdown 文件 + 模板 → Jekyll 处理 → 静态 HTML 网站
```

### Jekyll 的优势

- 学习曲线平缓
- 丰富的主题生态
- 与 GitHub 完美集成
- 支持 Markdown 写作

## 如何创建新文章

### 步骤 1：创建文件

在 `_posts` 文件夹中创建新文件，命名格式为：

```
YYYY-MM-DD-文章标题.md
```

例如：`2024-01-15-我的第一篇文章.md`

### 步骤 2：添加 Front Matter

在文件顶部添加 YAML 配置：

```yaml
---
layout: post
title: "文章标题"
date: 2024-01-15 12:00:00 +0800
categories: [分类 1, 分类 2]
tags: [标签 1, 标签 2]
---
```

### 步骤 3：撰写内容

在 Front Matter 下方使用 Markdown 撰写文章内容。

### 步骤 4：推送到 GitHub

```bash
git add .
git commit -m "添加新文章：文章标题"
git push
```

GitHub 会自动构建并更新您的网站！
